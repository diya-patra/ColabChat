About

ColabChat is a full-stack web application built using Flask and MongoDB, designed to showcase modern web development practices in a simple yet powerful way. It features secure JWT-based authentication, enabling safe user login and session management. With chat persistence, users can continue conversations seamlessly across sessions, while the Bootstrap-powered UI ensures a clean, responsive, and user-friendly experience. This project serves as a practical example for developers looking to understand the integration of backend services, database management, and frontend design into a cohesive application.
